API End Points

About

This website provides a RESTful API interface for data related to the Covid2019 Virus. Covid2019 is an active global emergency and the number of infected, recovered, and death cases are constantly changing. We will keep the data available through our API updated as this event unfolds.

End Points

We currently have 4 endpoints containing information about the Covid2019 outbreak:

  • Daily Infection Counts
    http://covid2019-tracker.appspot.com/api/date
  • Daily Infection Counts by Province/State
    http://covid2019-tracker.appspot.com/api/sars
  • Historical SARS Data
    http://covid2019-tracker.appspot.com/api/date
  • Important Headlines
    http://covid2019-tracker.appspot.com/api/timeline

Source

This data is sourced from the WHO (World Health Organization) and John Hopkins University. As news about the virus spread more people wanted access to this data, an open source collection of data was created.
